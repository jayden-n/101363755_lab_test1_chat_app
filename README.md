# Chat Buddy
> [!NOTE]
> Basic real-time chat app with room choices


## Installation & Setup

1. **Clone Repository:**

    ```sh
    git clone https://github.com/jayden-n/101363755_lab_test1_chat_app.git
    ```
    
2. **Install depenencies:**
    
    ```sh
    cd 101363755_lab_test1_chat_app
    npm i
    ```

3. **Run application:**
    
    ```sh
    npm run dev
    ```
